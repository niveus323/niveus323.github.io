알고리즘의 요건

1. 완전성과 명확성

  • 수행결과의 순서가 완전하고 명확하게 명세되어야 한다.
  • 알고리즘이 지시하는대로 실행하면 의도한 결과가 얻어져야 한다.

2. 입력과 출력

  • 입력: 알고리즘이 처리해야 할 대상, 제공되는 데이터
  • 출력: 입력 데이터를 처리하여 얻은 결과

3. 유한성

  • 단계는 유한해야하며 마지막 단계는 종료여야 한다.
